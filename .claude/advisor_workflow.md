## 프로젝트별 에스컬레이션 추가 조건

전역 CLAUDE.md의 공통 ESC-001~005 외 이 프로젝트 전용 조건:

| 코드 | 조건 | 해당 시나리오 |
|------|------|--------------|
| `ESC-P01` | Claude API 비용이 단일 요청에서 예상의 3배 이상 | 토큰 폭발적 증가 감지 시 |
| `ESC-P02` | 원격 MariaDB 스키마 변경이 기존 데이터 손실 가능성 있을 때 | ALTER TABLE, DROP 등 |
| `ESC-P03` | 박스권 판단 알고리즘에서 동등한 설계안 2개 이상 | 스캔 로직 설계 시 |
| `ESC-P04` | AI 분석 결과가 실제 매수 판단에 직접 사용되는 로직 | 자동 매수 트리거 설계 시 |

## 주의: 자동 매수 금지 원칙

이 시스템은 **분석 도구**이며 자동 매수/매도 실행 기능을 포함해서는 안 된다.
AI가 매수를 "권장"하더라도 실제 주문 실행 코드는 절대 포함하지 않는다.
